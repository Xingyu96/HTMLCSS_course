Useful/Related Instructions in html:

Declare new Array: var myArray = new Array();

Set array items directly: myOtherArray = ["rain", "sunshine"];

Add Element to Array: myOtherArray.push("snow");

Remove element from Array: myOtherArray.splice(1,2);

Use splice() to insert element(s): myOtherArray.splice(2,0,"hail", "cloudy");

Display info on console: console.log(myArray);
